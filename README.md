# Halo Ring Defense Game

A halo themed text-based Java game where you take on the role of a Spartan soldier defending the Halo Ring from Covenant forces. Explore the ring, encounter different alien species with varying difficulty levels, and defeat enemies to secure the ring. Manage your health with collectible items and engage in strategic combat.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open a command prompt or PowerShell in the project directory
2. Compile the Java files: `javac -d bin src/*.java`
3. Run the game: `java -cp bin Main`

Instructions for using the software:

1. Enter your Spartan name when prompted
2. Select an action from the menu (Explore, View Stats, View Inventory, View Defeated Enemies, or Quit)
3. When exploring, you have a chance to find items or encounter aliens
4. During combat, choose to Attack, Use Item, or Run away
5. Defeat 5 enemies to win the game
6. Your health must stay above 0 to survive
